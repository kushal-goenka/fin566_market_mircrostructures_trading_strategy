Detailed Description

ITradeTracker tracks the performance of each trading exposure a strategy initiates.

The class calculates the net realized pnl from the time a position is opened to the time it is closed. By default a new trade ID is generated every time you increase your position size through a new order, whereas partial fills within an order count towards the same trade ID. When you place an order, you can set legging_in to true to group multiple back-to-back buy or sell orders into a single trade ID
