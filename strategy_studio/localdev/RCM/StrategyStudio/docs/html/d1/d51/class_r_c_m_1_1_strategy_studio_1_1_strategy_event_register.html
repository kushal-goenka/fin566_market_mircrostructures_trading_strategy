Detailed Description

This is the registration interface that strategies use to subscribe to events at the beginning of a new trading day in Strategy::RegisterForStrategyEvents

const Event *  parentEvent 
)
virtual

Allows a strategy to register a closure object that will receive event messages from the parent event

Note the object passed in will be deep copied via the interface's "clone()" method the first time it is registered on a processor, and the object's "event_listener_name()" function is used for identification, ie to determine whether or not the object is already registered. Subsequent registration requests will simply return a pointer to the object created during the initial registration.

Parameters
closure_objectThe object that wishes to receive event messages.
parent_eventThe event to send to the closure object
Returns
Pair containing the created child listener event and a pointer to the instance of the callback object that was registered; returns nulls if registration fails
