Detailed Description

ImbalanceEventMsg is an event message containing information about the imbalance quantities related to an exchange's price setting auctions.

double RCM::StrategyStudio::ImbalanceEventMsg::clearing_price ( ) const
inline

Closest indicative price where imbalance would be zero

Contains 'near price' for NASDAQ Contains 'clearing price' for NYSE Open imbalance and 'continuous book clearing price' for closing imbalance Contains 'indicative match price' for ARCA Book Multicast and XDP 2.1, was for a time called 'reference price' in ARCA Book XDP 1 spec Contains 'indicative price' for BATS Auction feed
