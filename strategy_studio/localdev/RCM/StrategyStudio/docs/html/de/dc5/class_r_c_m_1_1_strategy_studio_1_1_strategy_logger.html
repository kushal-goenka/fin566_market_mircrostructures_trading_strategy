Detailed Description

Provides a default logger for the strategy. This will log to a log file associated with the Processor the strategy gets assigned for. We do not create a log file per strategy, because it may be important to look at the interaction between the multiple strategies assigned to a Processor, and because in the presence of many strategy instances, a log per instance could overrun the c runtime library or the OS's file handle limit.
