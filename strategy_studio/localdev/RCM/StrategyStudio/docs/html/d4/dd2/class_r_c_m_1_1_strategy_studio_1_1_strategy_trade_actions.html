Detailed Description

This class functions as a wrapper for a TradeActions object. It's purpose is to simplify the TradeActions interface for strategies by passing any additional objects that the ExecutionHandler needs. Also risk checks are performed at this level. Note: This class must be modified whenever interface functions are added or changed in the TradeActions class.
