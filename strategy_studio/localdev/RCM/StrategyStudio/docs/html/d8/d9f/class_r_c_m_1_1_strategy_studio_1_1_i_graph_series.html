Detailed Description

Interface for contributing data to a graphable series

Currently only time series graphing supported, and the timestamp is synched to the strategy's timekeeper To avoid overloading the network and Strategy Manager with data, data points are only sent to the GUI when their timestamps are newer than CUSTOM_GRAPH_THROTTLING_SEC seconds since the time of the last sent value. The throttling interval defaults to 1 second for a live server and 15 seconds for a backtesting server
